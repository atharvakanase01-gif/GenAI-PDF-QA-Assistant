# 🤖 GenAI PDF Q&A Assistant

An AI-powered PDF Question Answering Assistant built using Python, Streamlit, LangChain, FAISS, and Ollama.

The application allows users to upload PDF documents and interact with them using Local Large Language Models (LLMs) through Retrieval-Augmented Generation (RAG).

---

# 🚀 Features

✅ Upload PDF Documents  
✅ AI-powered Question Answering  
✅ Local LLM Integration using Ollama  
✅ Vector Search with FAISS  
✅ Retrieval-Augmented Generation (RAG)  
✅ Interactive Streamlit Dashboard  
✅ PDF Analytics  
✅ Chat History  
✅ Modern UI/UX

---

# 📊 Project Workflow

1. Upload PDF document
2. Extract PDF text
3. Split text into chunks
4. Generate vector embeddings
5. Store vectors in FAISS
6. Retrieve relevant context
7. Generate AI answers using local LLMs
